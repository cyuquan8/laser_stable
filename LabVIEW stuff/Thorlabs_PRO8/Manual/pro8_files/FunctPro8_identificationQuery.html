Pro8_identificationQuery

ViStatus Pro8_identificationQuery (ViSession instrumentHandle, ViChar _VI_FAR manufacturerName[], ViChar _VI_FAR instrumentName[], ViChar _VI_FAR instrumentSerialNumber[], ViChar _VI_FAR firmwareRevision[]);

Purpose

This function returns the Instruments Identification Strings.

Parameters

Input
Name Type Description
instrumentHandle ViSession

This control accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Default Value:  None

Output
Name Type Description
manufacturerName ViChar[]

This control returns the Manufacturer Name.
Note:
The array must contain at least 256 elements ViChar[256]. You may pass VI_NULL if you don't need this value.

instrumentName ViChar[]

This control returns the Instrument Name.
Note:
The array must contain at least 256 elements ViChar[256]. You may pass VI_NULL if you don't need this value.

instrumentSerialNumber ViChar[]

This control returns the Instrument Serial Number.
Note:
The array must contain at least 256 elements ViChar[256]. You may pass VI_NULL if you don't need this value.

firmwareRevision ViChar[]

This control returns the Instrument Firmware Revision.
Note:
The array must contain at least 256 elements ViChar[256]. You may pass VI_NULL if you don't need this value.

Return Value

Name Type Description
status ViStatus

This control contains the status code returned by the function call.
For Status Codes see function <Error Message>.
