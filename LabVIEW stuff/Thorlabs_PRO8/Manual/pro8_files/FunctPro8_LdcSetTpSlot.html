Pro8_LdcSetTpSlot

ViStatus Pro8_LdcSetTpSlot (ViSession instrumentHandle, ViInt16 slot);

Purpose

This function selects a TEC module for temperature protection.

Parameters

Input
Name Type Description
instrumentHandle ViSession

This control accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Default Value:  None

slot ViInt16

This control specifies the slot number of the TEC module.
Default Value: 1

Return Value

Name Type Description
status ViStatus

This control contains the status code returned by the function call.
For Status Codes see function <Error Message>.
