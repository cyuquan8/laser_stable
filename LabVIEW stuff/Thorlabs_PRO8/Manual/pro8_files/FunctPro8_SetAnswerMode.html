Pro8_SetAnswerMode

ViStatus Pro8_SetAnswerMode (ViSession instrumentHandle, ViBoolean answerMode);

Purpose

This function sets the Answer Mode of the instrument.

Parameters

Input
Name Type Description
instrumentHandle ViSession

This control accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Default Value:  None

answerMode ViBoolean

This control specifies if the instrument answers in Full Mode or in Value Mode.
Valid Range:
VI_OFF (0) - Full Mode (Default Value)
VI_ON  (1) - Value Mode
Note:
Use Value Mode if you want to increase speed.

Return Value

Name Type Description
status ViStatus

This control contains the status code returned by the function call.
For Status Codes see function <Error Message>.
