Pro8_ItcGetThCalSH

ViStatus Pro8_ItcGetThCalSH (ViSession instrumentHandle, ViInt16 values, ViPReal64 c1Value, ViPReal64 c2Value, ViPReal64 c3Value);

Purpose

This function returns the calibration values of the thermistor (Steinhart-Hart method).

Parameters

Input
Name Type Description
instrumentHandle ViSession

This control accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Default Value:  None

values ViInt16

This control selects the return values.
Valid Range:
0 - minimum Values
1 - maximum Values
2 - set Values (Default Value)

Output
Name Type Description
c1Value ViReal64 (passed by reference)

This control returns the C1-value (Steinhart-Hart parameter 1).

c2Value ViReal64 (passed by reference)

This control returns the C2-value (Steinhart-Hart parameter 2).

c3Value ViReal64 (passed by reference)

This control returns the C3-value (Steinhart-Hart parameter 3).

Return Value

Name Type Description
status ViStatus

This control contains the status code returned by the function call.
For Status Codes see function <Error Message>.
