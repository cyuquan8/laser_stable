Pro8_ItcGetThCalExp

ViStatus Pro8_ItcGetThCalExp (ViSession instrumentHandle, ViInt16 values, ViPReal64 bValue, ViPReal64 rValue, ViPReal64 tValue);

Purpose

This function returns the calibration values of the thermistor (exponential method).

Parameters

Input
Name Type Description
instrumentHandle ViSession

This control accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Default Value:  None

values ViInt16

This control selects the return values.
Valid Range:
0 - minimum Values
1 - maximum Values
2 - set Values (Default Value)

Output
Name Type Description
bValue ViReal64 (passed by reference)

This control returns the B-value [K].

rValue ViReal64 (passed by reference)

This control returns the R-value [Ohm].

tValue ViReal64 (passed by reference)

This control returns the T-value [°C].

Return Value

Name Type Description
status ViStatus

This control contains the status code returned by the function call.
For Status Codes see function <Error Message>.
