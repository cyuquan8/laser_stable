Pro8_WdmCwdmGetLfModType

ViStatus Pro8_WdmCwdmGetLfModType (ViSession instrumentHandle, ViPInt16 LFModulationType);

Purpose

This function returns the waveform of the internal low frequency modulation.

Parameters

Input
Name Type Description
instrumentHandle ViSession

This control accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Default Value:  None

Output
Name Type Description
LFModulationType ViInt16 (passed by reference)

This control returns the waveform of the low frequency modulation.
Valid values are:
0 - Noise
1 - Pulse
2 - Sine
3 - Square
4 - Triangle

Return Value

Name Type Description
status ViStatus

This control contains the status code returned by the function call.
For Status Codes see function <Error Message>.
