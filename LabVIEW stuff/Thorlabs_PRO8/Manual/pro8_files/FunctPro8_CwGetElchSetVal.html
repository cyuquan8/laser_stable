Pro8_CwGetElchSetVal

ViStatus Pro8_CwGetElchSetVal (ViSession instrumentHandle, ViInt16 setParameter, ViPReal64 startValue, ViPReal64 stopValue);

Purpose

This function returns the start and stop value of the selected parameter for a electrical characterisation.

Parameters

Input
Name Type Description
instrumentHandle ViSession

This control accepts the Instrument Handle returned by the Initialize function to select the desired instrument driver session.
Default Value:  None

setParameter ViInt16

This control selects the set parameter for a electrical characterisation.
Valid Range:
0 - Power [dBm] (Default Value)
1 - Power [W]

Output
Name Type Description
startValue ViReal64 (passed by reference)

This control returns the start value of the selected parameter.

stopValue ViReal64 (passed by reference)

This control returns the stop value of the selected parameter.

Return Value

Name Type Description
status ViStatus

This control contains the status code returned by the function call.
For Status Codes see function <Error Message>.
